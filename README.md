# Nazarbayev University Portal Frontend

A modern React frontend for the Nazarbayev University portal, featuring news and events sections.

## Features

- Responsive header with navigation
- News section with grid layout
- Events section with vertical list
- Modern purple/lavender color scheme
- Mock data for development

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
